## Features

- Modern
- Responsive
- Fully Customizable
- Easy to use
- Dark Mode

## Build Setup

```bash
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run dev
# build for production and launch server
$ npm run build
# preview production
$ npm run preview
```
